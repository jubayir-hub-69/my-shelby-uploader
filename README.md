# 🎨 Shelby - Decentralized Asset Workshop

A professional decentralized Web3 dApp powered by the Shelby Protocol and **Aptos L1**. This platform enables digital creators to seamlessly customize, securely watermark, and mint digital assets directly on-chain, ensuring verifiable ownership.

## 🚀 Live Demo
[Live Link to Shelby Workshop](https://my-shelby-uploader.vercel.app/)

## ✨ Key Features
* **Web3 Authentication:** Fully integrated with Aptos Petra Wallet, including direct Faucet access for Testnet.
* **Advanced Workshop Studio:** Upload custom media, apply real-time typography, and auto-embed the encrypted 'SHELBY HOT SECURE' watermark.
* **Live Latency Matrix:** Real-time diagnostic tool comparing Aptos Testnet Node performance against legacy storage systems.
* **Decentralized Vault:** Manage minted assets, verify transaction hashes via the Aptos Explorer, and share directly to X (Twitter).

## 💻 Tech Stack
* **Frontend:** React, Next.js, Tailwind CSS
* **Blockchain:** Aptos L1, Aptos TypeScript SDK, Petra Wallet Adapter
* **Graphics:** HTML5 Canvas API
